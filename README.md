# Game "Guess My Number"
A game where you can guess a random number between 1 and 20  
  
Finished: Oct 12, 2022

## **Shortcuts:**
**Space** - again  
**Enter** - check  
**Up / Down** - increase / decrease  

## Play: 
https://apterousteen.github.io/Game-Guess-My-Number/
